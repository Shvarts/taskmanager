Taskmanager

android 2.2
Програма являє собою клієнт для прийому і відправки миттевих повідомлень 
з сервером який безпосередньо звязаний з іншими клієнтами.

Фукціонал передбачає: 
1) Реєстрація нових користувачів;
2) Обмін повідомленнями між користувачами;
3) Можливість додавати нових друзів;
4) Функція пошуку користувачів;
